# Arduino: Tiny-MemorySequence

光と音で記憶力を試すシンプルなゲームです。

## 概要

Sim*n 風の記憶ゲームです。4色のLEDとボタンを使用して遊ぶことができます。
ゲームを開始すると光と音のパターンを提示し、プレイヤーはそれを記憶して正しい順序でボタンを押します。
パターンは段階的に長くなり、選択した難易度により最大8、14、20、31ステップのいずれかまで挑戦できます。

## 部材

| 種別 | 型番など | 数量 |
|:-----|:---------|:-----|
| ブレッドボード | 170H | 1個 |
| ジャンパーワイヤ | - | 8本程度 |
| タクトスイッチ | 2P 6x6mm | 4個 |
| マイコン | ATtiny13 | 1個 |
| LED | 緑、赤、黄、青 | 各1個 |
| 抵抗器 | 200Ω | 4個 |
| 抵抗器 | 220Ω | 1個 |
| パッシブブザー | - | 1個 |

## 配線

マイコンのPORTBに以下を接続：

- PB0: 緑LED
- PB1: 赤LED
- PB2: 圧電ブザー
- PB3: 黄LED
- PB4: 青LED

各LEDには対応するタクトスイッチをプルアップ構成で接続します。

## 操作

1. マイコンに電源を供給すると、起動メロディーが再生されます
2. LEDが点滅したら、いずれかのボタンを押してゲームを開始します
   - **緑ボタン**: イージーモード（最大8ステップ）
   - **赤ボタン**: ノーマルモード（最大14ステップ）
   - **黄ボタン**: ハードモード（最大20ステップ）
   - **青ボタン**: エクストリームモード（最大31ステップ）
3. マイコンが光と音のパターンを提示します
4. 提示されたパターンと同じ順序でボタンを押します
5. 正解すると次のレベルに進みます（パターンが1つ長くなります）
6. 不正解の場合はゲームオーバーとなり、最初からやり直しです
7. 選択した難易度のステップ数をクリアすると、ファンファーレが流れてゲームクリアです

## 注意

電池が消耗すると動作が不安定になります。
